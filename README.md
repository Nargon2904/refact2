# refact2

## Анализ
Функция FindMaxSubarray реализует поиск максимальной суммы подмассива.

Алгоритм выполняет 3 вложенных цикла по N, таким образом количество операций ~ N^3 (кубическая сложность).

Текущая реализация:

-Внешний цикл i от 0 до n-1

-Средний цикл j от i до n-1

-Внутренний цикл k от i до j

Теоретическая оценка сложности: T(n) = O(n³)

## Улучшение

Решение: Алгоритм Кадане (Kadane's Algorithm) с сложностью O(n).
